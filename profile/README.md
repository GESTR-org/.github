## Project description

Projekt dotyczy stworzenia systemu umożliwiającego sterowanie prezentacją multimedialną za pomocą gestów użytkownika. System wykrywa i interpretuje zestaw zdefiniowanych gestów, takich jak przejście do kolejnego slajdu, powrót do poprzedniego slajdu czy wskazywanie elementów prezentacji.

Dodatkowo projekt rozszerzony jest o analizę działania modelu z wykorzystaniem metod XAI. Obejmuje ona badanie dwóch głównych aspektów: wpływu cech niezwiązanych bezpośrednio z gestem (np. koloru skóry czy ubioru) oraz odporności modelu na zakłócenia obrazu, takie jak szum, rozmycie czy zmiany oświetlenia.
